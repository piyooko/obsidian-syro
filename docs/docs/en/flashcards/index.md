# Flashcards Overview

![Flashcard Deck Tree](../../assets/media/en/flashcard-decks-1.jpg)

## Module Overview
The flashcard module is the core place where Syro applies **Active Recall** and **Spaced Repetition**.

Unlike traditional standalone spaced-repetition software, Syro's flashcards are embedded directly inside your Obsidian notes. That means you can perform atomic knowledge tests while still being only one click away from the original Markdown source and its richer context.

This section presents the full workflow loop, from knowledge extraction to structured review.

## Core Workflow at a Glance
Syro gives you flexible paths into flashcard review, so you can choose the mode that fits your current focus:
- **Global view**: run `Review flashcards from all notes` to open the panoramic deck tree that includes every due card in your vault.
- **Focused review**: while reading a specific note, run `Review flashcards in this note` to review only the cards from the current document.
