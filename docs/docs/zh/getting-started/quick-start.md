# 5 分钟快速上手指南

> *注：本页将引导您完成 Syro 的最小工作流体验。我们建议您跟随步骤实际操作一次，这比阅读任何复杂的说明书都更为有效。*

## 模块导读
面对一个功能丰富的插件，最好的学习方式是从一个最简单的闭环开始。本页将向您展示如何在 5 分钟内，将一篇普通的 Obsidian 笔记纳入复习系统，并完成您的第一次“主动回忆”测试。

## 步骤一：追踪您的第一篇笔记（渐进阅读）
如果您有一篇需要长线阅读或复习的文章：
1. 在 Obsidian 中打开一篇您想要学习的笔记。
2. 唤出 **命令面板（Command Palette）**（通常通过 `Ctrl/Cmd + P`），搜索并执行 `Syro: Track this note`（追踪当前笔记）。
3. 搜索并执行 `Syro: Open Notes Review Queue in sidebar`。
4. 此时，您将在右侧侧边栏的“Today（今天）”或“New（新笔记）”分区中看到这篇笔记。它已经成功进入了您的渐进阅读队列。

## 步骤二：制作您的第一张闪卡（知识提取）
让我们在这篇笔记中提取一个具体的知识点：
1. **制作问答卡（Q/A）**：在笔记的空白处，输入一个问题，用 `::` 隔开，再输入答案。
   *示例：`Obsidian 的核心理念是什么？:: 是一个本地化、基于纯文本的知识库。`*
2. **制作填空卡（Cloze）**：选中笔记中的某句重点，使用 Obsidian 自带的高亮快捷键（`==文本==`）将其包裹。
   *示例：`Syro 是一个基于 ==间隔重复== 算法的复习插件。`*

## 步骤三：启动您的第一次复习会话
卡片制作完成后，让我们来测试一下大脑的提取能力：
1. 在命令面板中，执行 `Syro: Review flashcards in this note`（复习当前笔记中的闪卡）。
2. 此时，屏幕上将弹出复习界面。您会看到刚才写下的问题，或被遮挡的高亮词。
3. **在脑海中回忆答案**，然后按下 `空格键` 或点击“显示答案”。
4. 根据您回忆的顺畅程度，点击底部的 `Again`、`Hard`、`Good` 或 `Easy` 按钮。

**恭喜！** 您已经完整体验了 Syro 的核心闭环。系统底层的调度算法已经记录了您的反馈，并为您安排了这篇笔记与这几张卡片的下一次复习时间。

## 常用入口建议
为了后续使用的便利，您可以通过以下方式快速触发 Syro：
- **状态栏（Status Bar）**：在 Obsidian 窗口的最右下角，通常会显示待复习的笔记和闪卡数量，点击即可快速进入。
- **右键菜单**：在文件列表或当前笔记的标签页上点击右键，可以快速执行追踪或取消追踪。
