# 管理复习与心流

![牌组树界面](../../assets/media/en/review-deck.jpg)

## 模块导读
当您在库中积累了大量闪卡后，如何科学地调配每天的复习精力便成了首要问题。本页整合了**牌组树总览**、**复习会话操作**以及**牌组选项控制**的核心逻辑。

如果您希望了解如何解读系统给出的复习任务、如何在测试时顺畅操作，以及如何防止每天的复习量过载，请参考本页内容。

## 1. 牌组树：您的认知仪表盘
通过命令 `Review flashcards from all notes`，您可以唤出牌组树（Deck Tree）。在这里，您的卡片按目录或标签被归类，并展示三种状态的数字：
- **New（新卡）**：待开拓的处女地，尚未开始学习。
- **Learn（学习中）**：处于短期记忆脆弱期，系统会在今天高频推送以防遗忘。
- **Due（待复习）**：已进入长期记忆，且在今天恰好达到遗忘临界点，亟需巩固。

*提示：您可以随时折叠不需要关注的父级牌组，或者通过单篇笔记触发复习，将注意力极度聚焦在特定领域。*

## 2. 复习会话：主动回忆的竞技场
点击任意包含数字的牌组即可进入复习会话。在这个界面中，流畅的交互是保持心流的关键：
- **显示答案**：阅读卡片正面并在脑海中检索答案后，按下 `空格键`。
- **状态评分**：
  - `1 - Again (忘记)`：完全想不起来。卡片将退回重学。
  - `2 - Hard (困难)`：费了很大力气才回忆起。
  - `3 - Good (良好)`：正常回忆。这是最常用的标准反馈。
  - `4 - Easy (简单)`：脱口而出。系统将大幅推迟下次复习。
- **回溯原文**：若对卡片产生疑惑，可通过菜单或快捷键一键在后台打开该卡片所在的原始笔记，重温完整语境。

## 3. 牌组选项：控制每日认知负荷
人的意志力是有限的。如果某个牌组每天涌现几百张卡片，很容易引发挫败感。通过点击牌组右侧的齿轮（⚙️）图标，您可以调整**牌组选项**：
- **每日新卡上限（New Cards/Day）**：控制每天最多引入多少张尚未学习的新卡。调低此数值是缓解长期复习压力的最有效手段。
- **每日复习上限（Reviews/Day）**：控制每天最多展示多少张待复习的旧卡。通常建议保持较高数值，以优先捍卫已有记忆。

## 维持数据同步
日常的少量笔记修改，Syro 会通过后台的**自动增量同步**静默捕获。但如果您进行了大规模的重命名、移动文件夹或大批量删除操作，建议在命令面板执行一次 `Syro: Rebuild Cache`（重建缓存），以确保算法的视角与您实际的文件状态绝对对齐。
