# 优雅地编写闪卡 (Card Authoring)

![闪卡示例](../../assets/media/en/flashcard-cloze-example.jpg)

## 模块导读
将笔记转化为可测试的闪卡，是构建个人间隔重复系统（SRS）的关键一步。Syro 的设计哲学是**“零阻力结晶”**——您不需要离开当前的写作心流，只需使用最自然的 Markdown 标记，即可在行内瞬间生成卡片。

本页将向您介绍 Syro 支持的几种主要卡片格式。如果您想了解如何将平铺直叙的文本转化为高效的测试单元，请参考本指南。

## 1. 问答卡 (Q/A Cards)
问答卡是最经典、最稳定的闪卡形式，非常适合用于名词解释、概念辨析或公式背诵。

- **单行问答**：使用默认的分隔符 `::`。
  ```markdown
  什么是测试效应（Testing Effect）？:: 指通过主动提取记忆来显著提升长期保持率的现象。
  ```
- **多行问答**：如果答案较长或包含列表、图片，可以使用多行分隔符（默认如 `?`，或可在设置中自定义）。
  ```markdown
  列举渐进阅读的三个核心优势？
  ?
  1. 消除认知疲劳
  2. 提升元记忆判断
  3. 激发跨学科联想
  ```

## 2. 填空卡 (Cloze Deletions)
填空卡（也称挖空卡）是 Syro 核心功能。它允许您保留原文的丰富上下文，仅将核心概念遮挡。

- **基于高亮 / 粗体**：
  只要在设置中开启了对应选项，您可以直接使用 Obsidian 的高亮 `==文本==` 或粗体 `**文本**` 语法。
  ```markdown
  线粒体被称为细胞的==动力工厂==，它负责产生生命活动所需的 ATP。
  ```
  *(复习时，这张卡片的“动力工厂”将被遮挡，并在您点击后显示。)*

- **命令快捷键（多级挖空）**：
  当一段话中有多个需要测试的知识点时，您可以选中文本，通过命令 `Syro: Create Cloze (Same Level)`（同级挖空）或 `Syro: Create Cloze (New Level)`（新级挖空）快速生成标准的 `{{c1::文本}}` 格式。

## 3. 进阶卡片格式
对于有特殊排版需求的用户，Syro 同样提供了强大的解析支持：
- **表格与代码块**：您可以直接在答案区使用 Markdown 表格或代码块，系统会完整保留其格式并在复习界面中优雅渲染。
- **LaTeX 数学公式**：对于理工科用户，Syro 完整支持在问题、答案或挖空中嵌套 `$$` 包裹的 LaTeX 公式，确保公式在复习时的精准显示。

## 最佳实践建议
- **原子化原则（Atomic）**：一张卡片最好只测试一个核心概念。如果一张卡片的答案长达两三百字，您在复习时会感到极其痛苦且难以客观评分。
- **保留上下文**：如果卡片脱离了特定语境容易引起歧义，建议将卡片写在相关的笔记段落下方，复习时可随时“一键返回原文”查看背景。
- **统一语法风格**：虽然系统支持高亮、粗体、花括号等多种语法，建议您在个人的知识库中固定使用 1-2 种最顺手的标记，以保持笔记的整洁。
