# 笔记复习总览

![笔记复习队列](../../assets/media/en/note-review-queue.jpg)

## 模块导读
笔记复习（Note Review）模块是 Syro 应用“渐进阅读（Incremental Reading）”机制的核心载体。

与闪卡复习测试零碎知识点不同，笔记复习的单位是**完整的 Markdown 文件**。面对海量的长篇文献、待读网页或厚重的学习材料，您往往无法一次性将其消化。本模块的功能旨在帮助您将这些长线任务打散，依靠算法智能地分发到未来的碎片时间中，并在多次的接触中逐步完成理解与提炼。

## 核心工作流概览
为了在 Obsidian 中实现渐进阅读，Syro 提供了一套围绕“队列”与“上下文”的管理工具：
- **建立待读队列**：通过对笔记执行“追踪（Track）”操作，将其移交至底层的调度算法（如 WMS）接管。
- **分配阅读精力**：利用右侧的复习队列侧边栏，系统会为您排序出今天最值得阅读的笔记。
- **保留阅读状态**：通过特有的 Timeline 机制，系统会在您中断阅读时记录当前的滚动位置，确保下次重温时能无缝衔接。

## 章节导航
为帮助您建立有序的长篇阅读习惯，请参阅以下详细指南：

1. **[掌控笔记复习队列 (Queue Management)](./queue-management.md)**
   - 学习如何将单篇笔记或整个文件夹纳入/移出复习系统。
   - 了解如何解读侧边栏界面的时间分区（今天、逾期、未来等）。
   - 掌握如何通过调整标签过滤与优先级（Priority），干预算法排序，确保核心领域的知识始终被优先推送。

2. **[Timeline：保存阅读位置与历史进度](./timeline.md)**
   - 了解系统如何捕捉并记录您每一次阅读中断时的精确位置。
   - 学习如何通过历史时间线抽屉，快速回溯过去的阅读上下文，降低任务切换时的认知损耗。
