# 掌控笔记复习队列

![复习队列侧边栏](../../assets/media/en/note-review-queue.jpg)

## 模块导读
渐进阅读的魅力在于系统能帮您从浩如烟海的文献中，精准挑出今天最值得阅读的那几篇。本页将介绍如何追踪笔记、如何解读右侧队列侧边栏，以及如何通过标签和优先级重塑您的阅读秩序。

如果您需要在多篇长文中分配阅读精力，或希望将某个特定领域的学习材料置顶，请参考本页指南。

## 1. 纳入与移出调度体系
系统不会自动抓取您库里的所有文件，必须通过**追踪（Track）**操作明确授权：
- **就地追踪**：在阅读某篇笔记时，执行命令 `Syro: Track this note`，或右键选择对应选项。
- **批量追踪**：在左侧文件树中右键点击某个文件夹，选择追踪该目录下的所有笔记。
- 若某篇材料已失去时效性，执行**取消追踪（Untrack）**即可将其从调度队列中移除，此操作绝对不会删除您的本地物理文件。

## 2. 解读队列侧边栏
执行 `Syro: Open Notes Review Queue in sidebar` 可打开控制台。算法会将待读笔记按时间线进行折叠分组：
- **New（新笔记）**：刚追踪且尚未开始阅读的材料。
- **Overdue（逾期）**：之前错过或搁置的任务。不必焦虑，您可以随时拾起它们。
- **Today（今天）**：算法认为今天最适合您重新温习的笔记。
- *(单击侧边栏中的任意条目，即可在主编辑区打开该笔记的原文进行阅读。)*

## 3. 塑造阅读焦点：标签与优先级
当您的队列积压过多时，可以通过以下两种方式干预算法的排序：

- **标签筛选（Tags）**：
  在侧边栏顶部的过滤条中选择特定标签（如 `#心理学`），队列将瞬间收缩，仅显示该主题下的待读文章。您也可以在插件设置中配置“忽略标签（Ignored Tags）”，将包含如 `#草稿` 等标签的笔记永久剔除出视图。
  
- **调整优先级（Priority）**：
  同一天到期的笔记，谁排在最前面？在笔记条目上点击优先级修改按钮，赋予其一个更高的数值（如 80 或 100）。高优先级的笔记将获得算法的“插队特权”，确保高价值文献始终优先捕获您的注意力。

*(注意：阅读完毕后，请记得对该笔记执行“推进 / Review”操作，系统会将其移出今天的队列，并根据算法安排未来的重温日期。)*
